#  ExpenseTracker Pro – Personal Finance Manager

A full-featured expense tracker built with vanilla HTML, CSS, and JavaScript. All data stays in the browser’s local storage.

##  Features

-  Add expenses – description, amount, category, date  
-  Edit any expense – update details on the fly  
-  Delete individual expenses or clear all data  
-  Filter transactions by category and date range  
-  Live statistics – total (all time) + filtered total + transaction count  
-  Category breakdown chart – visual spending analysis (canvas)  
-  Persistent storage – data remains after closing/refreshing the browser  
-  Export to CSV – download all expenses as a spreadsheet  

##  Technologies Used

- HTML5 – semantic structure  
- CSS3 – modern gradients, flexbox, responsive design  
- JavaScript (ES6) – DOM manipulation, local storage, chart drawing  
- Font Awesome 6 – free icons for better UI
